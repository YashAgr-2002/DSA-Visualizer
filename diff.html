LINEAR SEARCH



1)In a linear search, the first element is used as the starting point, and until the element is not discovered, each element is compared to the first element.


2)Elements need not be sorted.


3)Any linear data structure, including an array, linked list, etc., can be used to implement the linear search.


4)It is focused on a sequential methodology.


5) It works better with smaller data sets and performs worse with bigger arrays.


6) Both a single-dimensional array and a multidimensional array can use it.

BINARY SEARCH




1)By locating the array's middle element, it determines the location of the searched element in the binary search algorithm.


2)Elements have to be sorted to perform binary search.(This is the pre-condition for binary search)


3)Binary search implementation is constrained because it can only be used with data structures that support two-way traversal.


4)It is based on the divide and conquer approach.


5) It works better with larger data sets due to it's higher efficiency.


6) It can only be used for single dimensional arrays.
