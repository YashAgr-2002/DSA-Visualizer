What is an ARRAY?

An array is a contiguous set of elements of the same data type stored in memory.

What is LINEAR SEARCH?

The process of traversing through the array while comparing the target element with each of those in the array.

What is BINARY SEARCH?

A search algorithm performed by dividing the array into half at every iteration(step).
